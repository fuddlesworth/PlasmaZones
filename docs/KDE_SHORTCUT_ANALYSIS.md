# KDE Default Shortcuts Analysis (from ~/.config/kglobalshortcutsrc)

## KDE/KWin uses (avoid conflict)

| Shortcut | Action |
|----------|--------|
| Meta+Alt+K | Switch to Next Keyboard Layout |
| Meta+Alt+L | Switch to Last-Used Keyboard Layout |
| Meta+Alt+S | Toggle Screen Reader (kaccess) |
| Meta+Alt+P | Move keyboard focus between panels |
| Meta+Alt+Arrow | Switch Window (Left/Right/Up/Down) |
| Meta+Ctrl+Arrow | Switch virtual desktop |
| Meta+Ctrl+Shift+Arrow | Move window to desktop |
| Meta+Ctrl+A | Activate Window Demanding Attention |
| Meta+Ctrl+Esc | Kill Window |
| Meta+Ctrl+X | Clipboard action popup |
| Meta+Ctrl+1-9 | (PlasmaZones: Snap to zone) |
| Meta+Ctrl+[ ] | (PlasmaZones: Rotate windows) |
| Meta+Shift+Left/Right | Move window to next/previous screen |

## PlasmaZones already uses

| Shortcut | Action |
|----------|--------|
| Meta+Alt+[ ] | Previous/Next layout |
| Meta+Alt+1-9 | Quick layout 1-9 |
| Meta+Alt+., | Cycle window in zone |
| Meta+Alt+Return | Push to empty zone |
| Meta+Alt+Esc | Restore window size |
| Meta+Alt+F | Toggle float |
| Meta+Alt+Shift+Arrow | Move window to zone |
| Meta+Alt+Shift+R | Resnap (current - hard to press) |

## Meta+Alt+{key} available (single letter, 2 modifiers)

- **Meta+Alt+R** — appears FREE (not in KDE or PlasmaZones)
- Meta+Alt+A,B,C,D,E,G,H,I,J,M,N,O,Q,T,U,V,W,X,Y,Z — also free

## Meta+Ctrl+{key} available (2 modifiers, easy)

- **Meta+Ctrl+R** — appears FREE
- Meta+Ctrl+Z, N, B (B = power profile), etc.

## Recommendation for Resnap

1. **Meta+Alt+R** — 2 modifiers, mnemonic (R=Resnap), one-handed. If user reports conflict, try:
2. **Meta+Ctrl+R** — 2 modifiers, different modifier set
3. **Meta+Alt+Z** — 2 modifiers, Z for "zone" layout
4. **Meta+Alt+N** — 2 modifiers, N for "new" layout
